Command line is too long.
发表于 2019-12-19| 更新于 2019-12-19|idea
字数总计: 88|阅读时长: 1 分钟|阅读量:

idea运行代码时出现

解决：
在.idea文件夹里的workspace.xml中的

<component name="PropertiesComponent">

标签下添加：

<property name="dynamic.classpath" value="true" />
